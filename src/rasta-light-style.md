# Яркая «Rasta Light» тема – стайлгайд

Версия: 1.0  
Автор: Vitaliy + AI‑art director

---

## 1. Концепция

Эта тема – дневная, солнечная, постерная версия растаманского мира Гайдука: открытый воздух, фестивальный плакат, комикс‑иллюстрация.  
Базовые опоры: плоские заливки, чистые контуры, раста‑флаг (зелёный‑жёлтый‑красный), психоделические сюжеты, но очень дружелюбный тон.

Главная задача: сохранить читаемость и простоту интерфейса, но ощущаться как яркая открытка / виниловая обложка.

---

## 2. Базовая палитра Rasta Light

### 2.1. Фоновые цвета

```text
Sunny Cream      #FFF5BF  – основной светлый фон (экраны, карточки)
Sunset Peach     #FFD27D  – градиентные переходы к краям, верхние баннеры
Cloud White      #FFFFFF  – всплывашки, модалки, "облака" под текстом
```

Sunny Cream – главный фон: тёплый, но достаточно нейтральный, чтобы не спорить с яркими акцентами.

### 2.2. Rasta‑акценты

```text
Rasta Red        #F94144  – красная полоса флага, акценты опасности/энергии
Rasta Yellow     #F9C74F  – жёлтая полоса, фон кнопок, светящиеся элементы
Rasta Green      #43AA8B  – зелёная полоса, основные CTA (play, запись)
Deep Forest      #136F4A  – тёмный зелёный для текста по зелёному/жёлтому
```

Основной action‑цвет: **Rasta Green**, поддерживаем его связкой с Rasta Yellow (фон) и Rasta Red (малые акценты).

### 2.3. Дополнительные цвета комикс‑мира

```text
Sky Blue         #7CC5FF  – небо, фон пустых экранов, иллюстрации
Psycho Orange    #FF9A3C  – солнца, вспышки, выделенные карточки
Ink Line         #222222  – основной контур и тёмный текст
Soft Gray        #8F8F8F  – второстепенный текст, системные подписи
```

## 3. Текст и нейтрали

```text
Text Primary     #222222  – основной текст на светлых фонах
Text Secondary   #5E5E5E  – подписи, описания
Text Invert      #FFFFFF  – текст на ярких Rasta‑заливках
Divider Light    #E3D8B8  – разделители, рамки блоков
Shadow Soft      rgba(0, 0, 0, 0.08) – мягкие тени под карточками
```

Типичные сочетания:
- Sunny Cream + Text Primary;
- Rasta Green / Rasta Red / Psycho Orange фон + Text Invert;
- Cloud White карточка + Divider Light + Shadow Soft.

## 4. Семантика и состояния

```text
Success          #43AA8B  – совпадает с Rasta Green
Warning          #F9C74F  – совпадает с Rasta Yellow
Error            #F94144  – совпадает с Rasta Red
Info             #7CC5FF  – Sky Blue
```

Состояния кнопок:
- **Primary (Play):** фон Rasta Green, текст/иконка Text Invert, лёгкая тень Shadow Soft.
- **Hover:** добавляем тонкий обводочный stroke `#0F5A3D` и усиливаем тень.
- **Pressed:** затемняем фон до `#2C8667`, уменьшаем тень.

Второстепенные кнопки ("поделиться", "добавить в плейлист") – фон Rasta Yellow или Cloud White с обводкой Rasta Green.

## 5. Градиенты и фоновые полотна

### 5.1. Флаг‑градиент для хедеров

```text
background: linear-gradient(90deg,
  #43AA8B 0%,   /* зелёный */
  #F9C74F 33%,  /* жёлтый */
  #F94144 66%,  /* красный */
  #F9C74F 100%
);
```

Использование: status‑бар, прогресс‑бар событий, специальные промо‑баннеры.

### 5.2. Солнечный градиент

```text
background: radial-gradient(circle at top left,
  #FFE66D 0%,
  #FFD27D 40%,
  #FFF5BF 100%
);
```

Использование: сплэш‑экран, фон экрана плеера.

## 6. Иллюстративный стиль

- Плоские заливки без сложных теней; максимум – небольшая мягкая тень под объектом.
- Контуры Ink Line (#222222) толщиной 2–3 px, слегка скруглённые углы.
- Персонажи и объекты – условно‑комиксные, с лёгким юмором, как на постерах к сказочным джемам.
- На границах блоков и секций можно использовать волнистую линию или «зубчики» как отсылка к психоделическим плакатам 70‑х.

## 8. Применение по зонам UI

### 8.1. Сплэш‑экран

- Фон: Солнечный градиент.
- Логотип и имя: Text Primary, возможно – в «облаке» Cloud White.
- Декор: волнистые полосы Rasta‑градиента по краям.

### 8.2. Главный список сказок

- Фон: Sunny Cream.
- Карточка: Cloud White c Shadow Soft и Divider Light по контуру.
- Название: Text Primary.
- Автор/мета: Text Secondary.
- Иконка play на карточке: круг Rasta Green, иконка белая.

### 8.3. Экран плеера

- Верх: флаг‑градиент тонкой полосой.
- Основной фон: Sunny Cream / Солнечный градиент.
- Обложка: в рамке Ink Line + лёгкий Shadow Soft.
- Прогресс‑бар: фон `#FFE6A3`, прогресс – Rasta Green, буфер – Rasta Yellow.

### 8.4. Мини‑плеер

- Фон: Cloud White, тонкая обводка Divider Light.
- Название трека: Text Primary.
- Кнопки prev/next: иконки Ink Line на прозрачном фоне; активное состояние – кружок Rasta Yellow.

### 8.5. Настройки / профиль

- Фон: Sunny Cream.
- Карточки секций: Cloud White + Divider Light.
- Тумблеры: активный – Rasta Green, выключенный – Soft Gray.

## 9. Чек‑лист для внедрения

1. Подключить `rastaLightTheme` как светлую тему, по умолчанию – использовать её для дневного режима.
2. Обеспечить переключение между тёмной «Rasta Night» и яркой «Rasta Light» темами.
3. Все primary‑действия – через Rasta Green, error/warning/info – через соответствующие раста/доп‑цвета.
4. Следить за контрастом текста на жёлтом и зелёном, при необходимости усиливать толщину шрифта.
5. Для иллюстраций использовать те же цвета палитры, избегать чужеродных холодных оттенков.

Дальше можно добавить отдельный гайд по иконкам и motion (например, лёгкие подпрыгивания и качание элементов, как у танцующих человечков на регги‑постерах).
