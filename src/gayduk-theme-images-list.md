# Список картинок для темы «Лунная даб-библиотека»

## Структура

Для темы используется иерархия ассетов: hero, splash, brand, covers, ambient-фоны, служебные картинки. Каждый пункт ниже содержит: имя файла, назначение и промт для генерации. Общий стиль: ночной архив аудиосказок, аналоговый звук, тёплый янтарный свет, тропический нуар, лёгкая психоделика, винтажная постерная фактура, без текста на изображениях.[cite:79]

## Рекомендации по генерации

- Не добавлять текст, логотипы и UI-элементы прямо на арт — это сделает набор более гибким.[cite:79]
- Сохранять единую палитру: тёплый янтарный, moss green, moon teal, deep soot, aged paper.[cite:79]
- Везде использовать схожий уровень grain и постерной фактуры, чтобы ассеты ощущались одной серией.[cite:79]
- Для media-session и app-icon проверять читаемость в размерах от 48×48 до 256×256.[cite:79]
